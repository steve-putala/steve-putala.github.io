Download .deb file from <https://www.virtualbox.org/wiki/Linux_Downloads>

Install using `apt`, change the path and file name

```bash
sudo apt install /tmp/virtualbox-7.0_7.0.12-159484~Ubuntu~jammy_amd64.deb
```
